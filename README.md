# Flooftaid FPS Game Documentation

## Overview
Flooftaid is an exciting first-person shooter game where players navigate through various levels, fighting enemies and completing missions.

## Features
- Engaging gameplay mechanics
- Multiple levels and environments
- Diverse weaponry and upgrades
- Multiplayer functionality

## Installation
1. Download the game from the official repository.
2. Extract the files to your desired directory.
3. Run the executable to start the game.

## How to Play
- Use `WASD` keys to move around.
- Use the mouse to aim and shoot.
- Follow on-screen instructions to complete the objectives.
